# 川渡りパズルアプリ

古典的な論理パズル「川渡り問題」を解くインタラクティブなWebアプリケーションです。

## 概要

船頭が船でネコ、ウサギ、野菜を左岸から右岸に運ぶパズルゲームです。船の容量と制約条件に従って、すべてのアイテムを安全に対岸に運ぶことが目標です。

## ゲームルール

- 目標：ネコ、ウサギ、野菜をすべて右岸に運ぶ
- 制約：
  - 船には1個まで（船頭は常に同乗）
  - 船頭がいない状態で、ネコとウサギまたはウサギと野菜を一緒にしてはいけない

## 開発コマンド

```bash
npm start    # 開発サーバー起動
npm test     # テスト実行
npm build    # 本番ビルド
```

## 機能

- インタラクティブなゲームUI
- 操作ログとCSVエクスポート
- ユーザーID管理
- 制約違反の自動チェック
